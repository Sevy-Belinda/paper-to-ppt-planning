# Page Structure Template

Use this template for the default markdown output.

```md
## 第 1 页

### 页标题

一句话说明本页在整场报告中的位置。

### 标题

页面主标题。

### 页面正文

- 保留真正会上页的主要内容
- 用适合口头讲解的表达
- 避免把这一部分写成论文摘抄

### 配图建议

- 需要图时说明图的类型、用途和摆放意图
- 如果本页不需要图，明确写“无”

### 图片来源

- 写明确来源链接、论文图号、或“用户提供”
- 如果没有图，写“无”

### 讲解提示

- 给讲者的过渡句、强调点、补充解释
- 这部分默认不上最终幻灯片
```

## Writing Rules

- `页面正文` 服务观众
- `讲解提示` 服务讲者
- `图片来源` 服务制作过程
- 一页只承担一个主要叙事任务
- 优先写出因果链，不要只堆名词

## Compression Rules

如果用户要求更短的输出：

- 保留 `页标题`
- 保留 `标题`
- 保留 `页面正文`
- 仅在关键页保留 `配图建议`
- 将 `图片来源` 和 `讲解提示` 只在必要页保留

## Deck Handoff Rule

当后续需要转成真正的 PPT 时，默认可上页内容是：

- `页标题`
- `标题`
- `页面正文`
- `配图`

默认不上页内容是：

- `图片来源`
- `讲解提示`
